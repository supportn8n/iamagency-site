## ВАЖНО — известное ограничение
get_figma_data иногда СХЛОПЫВАЕТ внутренности карточек (отдаёт пустыми группами — так было
с планшетными УТП/Тарифами). Обход: при сборке блока делать get_figma_data КОНКРЕТНОГО фрейма
(nodeId) — часто внутренности приходят. Если всё равно пусто — брать контент из десктопного
модуля и масштабировать, ПОМЕТИТЬ в этом файле «приближённо, не 1:1».

## Метод за один проход loop (1 единица)
1. Взять первый `[ ]` из чек-листа ниже.
2. get_figma_data нужного фрейма (Tablet и/или Mobile), собрать модуль(и).
3. Подключить через ResponsiveBlock. `npx next build`.
4. Проверить скрином (gstack browse, viewport 768 и 375), НЕ ломая десктоп.
5. Деплой (npx vercel --prod --yes → alias) + git commit/push.
6. Отметить `[x]` (или `[~]` если приближённо) + короткая заметка.
