Дата проверки: 2026-07-14

## Семантическая карта

KeywordTool MCP в активной конфигурации проекта не подключён: в `.mcp.json` доступны только Figma и Magic. Карта собрана по поисковой выдаче, действующему контенту `iamagency.su` и интентам новых посадочных страниц.

| Страница | Основной кластер | Поддерживающие запросы |
|---|---|---|
| `/` | SMM-агентство | SMM-продвижение, продвижение в социальных сетях, ведение соцсетей под ключ |
| `/uslugi/brendbuk-i-smm-strategiya` | разработка SMM-стратегии | брендбук для соцсетей, стратегия продвижения бренда |
| `/uslugi/vedenie-sotssetey` | ведение соцсетей под ключ | ведение ВКонтакте, Telegram и Instagram, контент-план для бизнеса |
| `/uslugi/marketing-i-prodvizhenie` | продвижение в социальных сетях | influence-маркетинг, посевы, таргетированная реклама |
| `/uslugi/kontent-syomki` | контент для соцсетей | съёмка Reels, мобильная съёмка, фото- и видеоконтент |
| `/marketing` | маркетинговое агентство полного цикла | performance-маркетинг, digital-маркетинг, интернет-маркетинг |
| `/keisy`, `/case/*` | SMM-кейсы | кейсы продвижения по нишам, портфолио SMM-агентства |
| `/shkola-smm` | курс SMM с нуля | обучение SMM, SMM-менеджер с нуля, школа SMM |
| `/blog`, `/blog/*` | блог о SMM и маркетинге | продвижение, визуал, нейросети, социальные сети |
