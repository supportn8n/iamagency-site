## Методология «1:1»
- Каждая секция = фрейм 1440×H, дети позиционируются абсолютно. Хелперы `P(x,y,w,h)` (%, по 1440×1024) и `F(px)` (cqw) уже есть в `src/app/page.tsx`. Использовать тот же подход.
- Извлечь из `/files` ВСЕ элементы (TEXT: characters+style.fontFamily/Weight/Size+fill; RECTANGLE/VECTOR/FRAME: x/y/w/h, fills, strokes, cornerRadius). Воспроизвести каждый.
- Шрифты: заголовки Coolvetica (нет кириллицы → выгружать текст-картинкой из Figma как сделали с Hero, либо плейсхолдер Oswald + в TODO), текст Inter. Иконки: Remix (react-icons `ri`) — Telegram/Instagram/WhatsApp точные; Max — в TODO (нет в библиотеках).
- Каждый блок — отдельный компонент `src/app/blocks/<Name>.tsx`, импортируется в `page.tsx` по порядку (избегаем конфликтов правок).
- Деплой: `cd iamagency-site && npx vercel --yes --prod` (CLI залогинен, проект iamagency-site). Прод-URL: https://iamagency-site.vercel.app
